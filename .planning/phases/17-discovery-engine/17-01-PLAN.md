```python
def encode_string(text: str, num_registers: int) -> list[int]:
    """Encode ASCII string into uint16 register list, null-padded."""
    raw = text.encode("ascii").ljust(num_registers * 2, b"\x00")
    return [int.from_bytes(raw[i:i+2], "big") for i in range(0, num_registers * 2, 2)]
```

    Create `src/venus_os_fronius_proxy/scanner.py` with these components:

    1. **decode_string(registers: list[int]) -> str**: Reverse of encode_string. Join registers as big-endian bytes, decode ASCII, strip nulls and whitespace.

    2. **@dataclass ScanConfig**: ports (default [502, 1502]), tcp_timeout (0.5), modbus_timeout (2.0), concurrency (15), scan_unit_ids (default [1]), skip_ips (default set()). Use __post_init__ for mutable defaults.

    3. **@dataclass DiscoveredDevice**: ip (str), port (int), unit_id (int), manufacturer (str), model (str), serial_number (str), firmware_version (str), supported (bool). The `supported` field is computed: True if "solaredge" in manufacturer.lower().

    4. **detect_subnet() -> IPv4Network**: Run `subprocess.run(["ip", "-j", "-4", "addr", "show"], capture_output=True, text=True, timeout=5)`. Parse JSON. Iterate interfaces, skip 127.x and 169.254.x. Return IPv4Network(f"{ip}/{prefix}", strict=False). Raise RuntimeError if no usable interface.

    5. **async _probe_port(ip: str, port: int, timeout: float) -> bool**: Use `asyncio.wait_for(asyncio.open_connection(ip, port), timeout=timeout)`. Close writer in try/finally. Return True on success, False on TimeoutError/OSError.

    6. **async _verify_sunspec(ip: str, port: int, unit_id: int, timeout: float) -> DiscoveredDevice | None**: Create AsyncModbusTcpClient(ip, port=port, timeout=timeout, retries=0). In try/finally (close in finally): connect, read_holding_registers(40000, count=2, slave=unit_id). Check registers == [0x5375, 0x6E53]. Read 67 registers at 40002. Parse Common Block:
       - DID at offset 0 (verify == 1)
       - Length at offset 1 (verify == 65)
       - Manufacturer: decode_string(regs[2:18])
       - Model: decode_string(regs[18:34])
       - Firmware: decode_string(regs[42:50])
       - Serial: decode_string(regs[50:66])
       Return DiscoveredDevice with supported = "solaredge" in manufacturer.lower()

    7. **async scan_subnet(config: ScanConfig | None = None, progress_callback: Callable | None = None) -> list[DiscoveredDevice]**: Main entry point. Detect subnet. Phase 1: TCP probe all hosts (skip skip_ips) on all ports with Semaphore(config.concurrency). Phase 2: Verify SunSpec on open hosts -- iterate unit_ids per host. Return list of DiscoveredDevice. Log with structlog at key points (scan.start, scan.probe_complete, scan.device_found, scan.complete).

    The progress_callback parameter signature: `callback(phase: str, current: int, total: int)` -- called during probe and verify phases. This is for Phase 20 to wire to WebSocket. Scanner does not import WebSocket.

    Use `from pymodbus.client import AsyncModbusTcpClient` (same import as solaredge.py).
