## Critical Integration Gaps

### Gap 1: Distributor not reachable from poll loop

**Impact:** THRT-08, THRT-09, THRT-12
**Root cause:** `AppContext` has no `distributor` field. `__main__.py` creates the distributor but doesn't set `app_ctx.distributor`.
**Fix:** Add `distributor: object = None` to `AppContext` in `context.py`; set `app_ctx.distributor = distributor` in `__main__.py` after line 141.

### Gap 2: DeviceRegistry._distributor never set

**Impact:** Throttle table Limit column, contribution bar throttle state
**Root cause:** `DeviceRegistry._distributor` declared but never assigned. `__main__.py` passes `registry` to distributor but not the reverse.
**Fix:** Add `registry._distributor = distributor` in `__main__.py` after creating the distributor.

### Gap 3: AGG-02 DC averaging

**Impact:** Mixed fleet DC voltage reading in virtual PV
**Root cause:** `aggregation.py` avg_keys includes `dc_voltage_v` without filtering devices with zero DC.
**Fix:** Skip devices where `dc_power_w == 0` from DC voltage averaging.
